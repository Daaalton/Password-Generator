# Password Generator

Generador de contraseñas implementado en python.

## Funcionamiento
Devuelve una contraseña pseudo-aleatoria de 18 caracteres, asegurándose de que contenga al menos una letra mayúscula, un número y un símobolo especial entre: '!', '@', '#', '$', '%', '^', '&', '*', '(', ')', '-', '_', '=', '+'.

## Uso
python passGenerator.py
